概念：設置cURL

在本主題中，您將學習cURL，這是一種對於測試和發出API請求有用的工具。 cURL是用於發出HTTP請求的命令行工具。我們在用於測試Brightcove API調用的shell腳本。如果您使用的是Mac MacOS或任何Unix / Linux系統，則已經具有cURL。如果您使用的是Windows，請參見以下獲取cURL的說明。

視窗

如果你有視窗，可以如下下載安裝cURL。

安裝cURL

  1. 在視窗 , 創建一個名為的新文件夾捲曲在您的 C: 驅動器中。

    C:curl

  2. 去https://cygwin.com/install.html按照那裡的說明，根據您的系統選擇32位或64位安裝程序。在安裝過程中，將提示您安裝軟件包。打開網絡包裝，然後檢查捲曲選項，然後完成安裝。

測試您的設置

跑步cURL命令，您只需鍵入curl（帶有要發出的請求的適當標誌）（在終端或DOS提示符下），然後按Return鍵。要確保正確安裝了cURL，請打開“終端"或DOS窗口，鍵入curl在提示符下，然後按Return鍵。您應該看到以下消息：

 curl: try 'curl --help' or 'curl --manual' for more information

傳送資料

如果您需要通過API請求發送數據（例如，在POST或PATCH請求中），則可以通過多種方法進行。您可以內聯或在單獨的文件中包含數據。這是一個播放器管理API示例，將數據作為單獨的JSON文件提交：

 curl --request POST
    --url https://players.api.brightcove.com/v2/accounts/57838016001/players
    --header 'authorization: Bearer YOUR_ACCESS_TOKEN_HERE'
    --header 'content-type: application/json'
    --data <player_data/player_config.json
注意“< " 符號，用於發送文件內容。使用它而不是“@”符號，它將文件附加為上傳。
