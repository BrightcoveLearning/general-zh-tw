簡介

Wireshark 是一個流行的開源應用程序，用於分析網絡協議的深度遠遠比你可以通過，例如，瀏覽器開發人員工具的網絡部分實現。這對於診斷網際網路應用程式 (例如 Brightcove Player) 或透過我們的 API 與 Brightcove 互動的應用程式等網際網路應用程式的問題非常有用。

即使您自己對網路知識不夠深入，無法完全瞭解您從 Wireshark 取得的資訊，擷取這些資訊對 Brightcove 支援部門診斷問題有很大幫助，我們的支援代表可能會詢問您是否可以從線鯊魚讓他們能更好地幫助你。

以下步驟將引導您完成 Wireshark 的基本安裝和使用。

安裝

  1. 如果尚未安裝Wireshark，請轉到https://www.wireshark.org/#download並下載與您的平台匹配的安裝程序。
  2. 執行安裝程式。如果您看到任何選項，請安裝所有可用的元件。(選項因發行版本和平台而異。)

使用線鯊魚

重要的：在開始之前，最好關閉盡可能多的訪問Internet的應用程序（例如：電子郵件客戶端，IM或社交媒體應用程序，Dropbox，Evernote和其他基於雲的存儲應用程序）。這將有助於將與您嘗試診斷的問題無關的網路傳輸的「噪音」降至最低。

  1. 發射線鯊
  2. 選擇一個捕獲過濾器-您想要的過濾器可能會顯示大部分或至少一部分流量：
  3. 雙擊您想要開始捕獲數據的過濾器：
  4. 數據捕獲運行時，通過加載播放器，運行應用程序或任何內容來重新創建您嘗試診斷的問題。
  5. 完成重現問題後，請按一下 Wireshark 中的 [停止擷取資料] 按鈕：
  6. 現在，點擊保存捕獲文件按鈕從 Wireshark 導出數據：
  7. 將檔案儲存在電腦上的預設格式。
  8. 在支援要求中將檔案交付給 Brightcove 支援，詳細說明您遇到的問題。
