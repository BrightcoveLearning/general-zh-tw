在移動設備上調試

在本主題中，您將了解如何在行動裝置上偵錯 JavaScript。在開發測試和調試移動平台上的內容可能是非常具有挑戰性的。

概述

在本文件中，您將學習如何開始在行動裝置上進行偵錯，並涵蓋下列解決方案：

  • 在 iOS 裝置上進行除錯
  • 在安卓裝置上進行除錯
  • 在行動裝置上使用查爾斯
本文檔可作為開發人員在移動設備上進行調試的起點。它不涵蓋移動調試的所有方法。有關一般調試的更多信息，請訪問JavaScript 調試基礎文檔。

iOS 除錯

iOS 除錯的最低需求

  • 在 Mac Mac 上運行的野生動物園版本 6 以上
  • 智慧型手機
注意：如果您使用的是 iOS 10.3 或更高版本，請打開設置並導航到常規>關於>證書信任設置，找到 Charles Proxy 證書，打開它，為它啟用完全信任。

使用 Safari 遠端除錯

蘋果的 iOS 6 更新推出了 Safari 遠程調試，它允許您在 iOS 設備上的 Safari 應用程序中調試網頁。若要開始使用，請依照下列步驟操作：

  1. 透過 USB 連接線將 iOS 裝置連接至電腦。
  2. 在您的設備上，打開設定值應用程序。選擇蘋果瀏覽器，向下滾動到頁面底部，然後打開先進的菜單。啟用網頁檢查器選項。
  3. 仍在您的設備上，打開蘋果瀏覽器應用程序並瀏覽到任何網頁。
  4. 在Mac上，啟動蘋果瀏覽器， 打開首選項（ Safari>首選項），然後導航到先進的部分。啟用在菜單欄中顯示“開發"菜單選項。
  5. 打開開發菜單。您將會看到列出您的 iOS 裝置。當您將滑鼠移到裝置名稱上時，您會看到您在裝置上瀏覽的網站 URL。
  6. 選擇URL和一個網頁檢查器窗口將打開，您可以從桌面預覽，檢查和調試移動瀏覽器中的內容。
  7. 在裡面資源部分，您可以預覽和檢查網頁。將鼠標懸停在網頁檢查器將在iOS設備上的瀏覽器中突出顯示相應的元素。
  8. 您也可以使用調試器和安慰部分，通過設置斷點，記錄數據等來進一步調試頁面。
有關如何使用 Safari Web Inspector 的更多詳細信息，請訪問蘋果的開發者文檔 .

安卓除錯

安卓調試的最低要求

  • 在 Mac 或視窗上執行
  • 安卓裝置與瀏覽器

注意：本節中的步驟是使用運行 Android 4.3 的設備編寫的。

在您的裝置上啟用除錯

您必須先在裝置設定中啟用 USB 偵錯，才能開始偵錯裝置上的內容。

  1. 如果您運行的是Android 4.2+，則需要啟用設備的開發人員選項在您打開USB調試之前。在您的設備上，打開設定值應用，然後選擇關於手機菜單。找到您設備的內部編號然後點擊七次
    如果您運行的是舊版本的 Android，您可以通過直接進入“設置”來啟用 USB 調試>開發者選項。
  2. 打你設備的後退按鈕，您將看到開發人員選項設置中列出的菜單。
  3. 打開開發人員選項菜單，然後選中相應的框以啟用USB調試。

遠程調試使用

現在您的裝置上已啟用 USB 偵錯，請依照下列步驟透過 Chrome 啟動遠端偵錯：

  1. 在您的計算機上，啟動鉻合金並瀏覽到about:inspect。在設備頁面上，啟用發現USB設備選項。
  2. 透過 USB 傳輸線將裝置連接至電腦。在您的裝置上，系統會出現一個對話方塊，詢問您是否要允許 USB 偵錯。選擇好的以允許您的計算機連接到您的設備。
    如果您使用的是 Windows 計算機，則可能需要安裝 USB 驅動程序。欲了解更多信息，請訪問Android 開發者文檔.
  3. 在您的設備上，打開鉻合金應用程序並瀏覽到網頁。
  4. 還給鉻合金在你的電腦。在about:inspect頁面上，您應該會看到已連接的設備。在裝置名稱下方，您可以在裝置上看到 Chrome 中開啟的所有網頁清單。
  5. 要開始調試網頁，請選擇檢查頁面網址下方的鏈接。這將打開Chrome的開發者工具。
  6. 在左邊元素標籤，您可以選擇設備圖標以打開DevTool的截屏功能。這可讓您開啟內容在行動裝置上顯示的即時預覽。有關如何使用屏幕廣播與設備進行交互的更多信息，請訪問Google的文件。
有關如何使用 Chrome 開發者工具的更多詳細信息，請訪問谷歌的開發者文檔 .

查爾斯移動

查爾斯是一個 Web 代理，可以在您自己的計算機上運行，允許您記錄通過您的 Web 瀏覽器發送和接收的數據。造訪查爾斯網站有關此應用程序的更多信息。

查爾斯的最低要求

  • 查爾斯與視窗，Mac 和 Linux 兼容
  • iOS 及/或安卓裝置

在您的機器上設置查爾斯

按照以下步驟讓查爾斯在您的電腦上運行。

  1. 下載並在您的計算機上安裝Charles。
  2. 在您的計算機上運行查爾斯。從菜單中選擇代理>代理設置。確保已分配端口（例如8888），並且啟用透明的HTTP代理選項被選中。
  3. 選擇好的保存更改並關閉對話框。
  4. 從菜單中選擇代理> SSL代理設置。確保啟用SSL代理選項被選中。
  5. 在裡面地點部分，選擇添加按鈕。在裡面修改位置對話框中，導航到港口字段並輸入443。選擇好的保存此更改。
  6. 選擇好的關閉修改位置對話。
  7. 選擇好的退出SSL代理設置對話。
  8. 導航幫助> SSL代理>在移動設備或遠程瀏覽器上安裝Charles Root證書。
  9. 您應該會看到一則訊息，識別您將用來設定裝置的本機伺服器 IP 和連接埠號碼 (8888)。您還應該看到 URL 來下載並安裝證書。將這些檔案複製到電腦以供日後使用。
    請注意，您需要啟用要信任的 Charles 證書。

在您的裝置上設定查爾斯

在本節中，您將學習如何設置您的 iOS 和/或 Android 設備，以便在您的計算機上與查爾斯一起工作。

設定您的 iOS 裝置

  1. 在您的裝置上，確定您已連線到與電腦相同的網路。您可以通過導航到來驗證這一點設置> Wifi並選擇網絡名稱以顯示高級選項。
  2. 在底部的HTTP部分中，選擇手動的模式。然後，新增伺服器 IP 和連接埠號碼 (8888) 您從查爾斯在上一個步驟中複製。
  3. 在您的設備上，打開瀏覽器並導航到http://www.charlesproxy.com/getssl。您將被重定向到設備設置，系統將提示您安裝Charles Proxy自定義根證書輪廓。選擇安裝按鈕可在您的設備上安裝Charles證書。
  4. 請依照下列步驟安裝查爾斯憑證。完成後，選擇完畢。
  5. 接下來，您已準備好確保您的裝置信任 Charles 憑證。在您的設備上，導航至常規>關於>證書信任設置。在這裡你應該找到查爾斯代理證書。
  6. 啟用查爾斯憑證。請注意警告訊息。您可能想要在不使用查爾斯進行偵錯時停用此憑證。
  7. 現在，你將能夠看到在查爾斯發送和接收的任何數據。

設定您的安卓裝置

這些步驟僅適用於運行 Android 2.3 及更高版本的設備。
  1. 在您的設備上，打開瀏覽器並導航到//chls.pro/ssl。
  2. 系統將提示您證書名稱對話。為證書指定名稱並設置用於到無線上網。選擇好的安裝證書。
  3. 接下來，打開設置> Wi-Fi菜單。您會在裝置上看到已連線的網路清單。選取您目前連線的無線網路。請記住，這個網路必須符合您在電腦上連線的網路。
  4. 這將打開一個包含三個選項的小對話框-編輯，查看或忘記。選擇編輯。
  5. 在此對話框中，向下滾動到底部並啟用顯示高級選項場地。這將顯示代理和 IP 設置。
  6. 改變代理設定到手動的。
  7. 填寫代理主機名字段以及您計算機的IP地址和代理端口字段以匹配在Charles中設置的端口（默認情況下，端口設置為8888）。選取「 儲存」 。
  8. 在裝置的預設瀏覽器中，瀏覽至網站。您將能夠看到在查爾斯發送和接收的任何數據。
