Brightcove 產品概述

本主題提供 Brightcove 提供的產品簡要摘要。

布萊特灣文件是按產品組織的。Brightcove還提供產品包，例如視頻營銷套件或者企業視頻套件其中包含一個或多個產品。使用文件時，您應該搜尋正在使用的特定產品，而不是搜尋產品套件。

視訊雲端

Brightcove 視頻雲是一個多功能於一身的視頻託管和發布平台，輕鬆觸及您的觀眾。視訊雲端可讓您輕鬆上傳、管理和發佈影片。輕鬆建立播放器樣式，並檢視您帳戶中影片的表現。

學到更多關於視頻雲。

畫廊

圖庫讓您能夠提供身歷其境的視訊體驗。您可以在幾分鐘內建立自訂互動式視訊體驗、品牌入口網站、以視訊為中心的登陸頁面和即時串流活動網站。

學到更多關於畫廊。

布萊特灣玩家

Brightcove 播放器是一個高性能的跨平台 HTML5 第一視頻播放器，快速載入，跨桌面和行動平台提供高品質的自適應位元率 HLS 視頻，並與您最喜愛的廣告和分析系統整合。

主要功能

  • HTML5優先 -新的Brightcove Player的UI完全用HTML5編寫，並且僅在較舊的瀏覽器上依賴Flash，從而在移動設備和台式機設備上實現了一致的外觀和行為。
  • 反應靈敏 -默認情況下，默認的播放器外觀是響應式的，並且會自動調整大小以適應不斷變化的Web佈局，同時保持視頻的縱橫比。
  • 快速加載 -即時加載並以極快的速度播放第一幀圖像，觀眾可以立即欣賞到您的內容。
  • 輕鬆定制使用標準的網絡技術（CSS和JavaScript）
  • 跨平台HLS -即使在不具備本地HLS功能的平台上，也可以提供高質量的自適應比特率視頻-以最少的下載量提供最佳的質量。
  • 球員管理 -通過Video Cloud或新的Player Management API輕鬆跟踪播放器配置並更改樣式和插件。輕鬆更新您網站上的玩家，而不需更改每個頁面的代碼。
  • 廣告和Analytics（分析）整合 -Brightcove Player提供了許多插件，可與流行的廣告和分析系統集成，包括Google Doubleclick，Freewheel等。

學到更多關於Brightcove Player。

生活

直播模組可用來協助您在桌上型電腦和行動裝置上直播活動。在直播模組中輸入您的活動資訊後，系統會提供一組編碼器設定，讓您插入現場編碼器。所有轉碼都在雲端完成，因此您不需要基於硬件的轉碼器。

學到更多關於實時模塊。

社交

Brightcove 社會允許您輕鬆地發布視頻在您的視頻雲帳戶出流行的社交媒體平台，如 YouTube，臉書和推特。視訊可以手動發佈或使用「自動同步」。「自動同步」是用來根據與視訊相關聯的標籤，自動將視訊同步至社交媒體頻道。

學到更多關於社交模塊。

受眾

觀眾模組將從布萊特灣的個別視頻觀看分析連接到甲骨文埃洛誇，市場，HubSpot 和其他行銷自動化平台。此外，我們的 Audience API 可讓合作夥伴和客戶擷取相同的地圖相關資料，以便傳遞至其選擇的任何資料儲存或處理平台。在這兩種情況下，Brightcove 都能讓數位行銷人員衡量視訊的影響，以便在個人化層面吸引受眾，並推動轉化率。

學到更多關於“受眾群體"模塊。

禪編碼器

Brightcove Zencoder 提供快速、簡單且功能強大的雲端視訊轉碼解決方案，可輕鬆根據您的需求進行擴充。透過 Zencoder 的視訊編碼 API，您將無縫整合您的應用程式與我們極快且可擴充的編碼平台，而且不需要幾個星期。我們記錄良好的 API、易於使用的 API 要求建置器、整合程式庫和專屬支援，簡化了簡單的解決方案流程。

學到更多關於Zencoder。

SSAI

Brightcove 的 SSAI 技術會將廣告動態地拼接到伺服器上的串流中，讓內容和廣告都能以單一連續串流形式傳送。這種方法可讓您將廣告投放到任何可以播放影片的裝置、消除緩衝區和其他廣告插入成品以改善觀眾的體驗，並透過管理廣告攔截器來提高收益。

學到更多關於SSAI。

OTT 流程

由 Cogotido 提供支援的 Brightcove OTT Flow，降低了推出美觀的多平台 OTT 服務所帶來的技術複雜性和進入市場障礙。布萊頓灣視頻雲通過業界領先的視頻播放器，提取，轉碼，元數據管理和SDK為OTT Flow的骨幹提供支持。Prodido Studio Pro 可從中央雲端儀表板降低維護成本、提供引人入勝的用戶體驗，以及對跨平台的 OTT 服務進行全面管理。
