    布萊特灣開源項目

    以下是所有目前 Brightcove 技術開放原始碼專案的清單，按類別分類。
    開源項目沒有被布萊特灣正式支持。

    歡迎來到布萊特灣開放原始碼您可以在這裡找到由 Brightcove 員工或布萊特灣社區成員建立的許多項目，以協助加速您的布萊特灣發展。

    如果您有興趣為項目做出貢獻，請前往 Github 上的個別項目網站。如果您有想要在此處列出的項目，聯繫我們。

    目前的 API

    1. API 要求疑難排解指令碼

      疑難排解布賴特灣 API 的 HTTP 要求和回應。

      https://github.com/BrightcoveOS/api-request-troubleshooting-script

    2. 布萊特灣上傳者

      用於直接多部分上傳到 s3 的前端模塊，用於布萊特灣攝取的秘密醬汁。

      https://github.com/BrightcoveOS/evaporate-brightcove

    3. 布萊特灣播放 API 包裝

      這是用於Brightcove播放API特蕾莎·紐曼（Theresa Newman）撰寫。

      https://github.com/BrightcoveOS/Brightcove-Playback-API-Wrapper

    4. 用於布萊特灣動態攝取的 PHP 包裝

      該項目為整合Brightcove提供了一個起點Dynamic Ingest API進入您的應用程序。它提供了簡單的方法，使用提取式內嵌或來源檔案上傳來新增和更新視訊和相關媒體。

      https://github.com/BrightcoveOS/php-wrapper-dynamic-ingest

    5. API 外觀

      這個謙虛的圖書館旨在成為布萊特灣的服務器 API 上的一個簡單的外觀。由於黑幫需要額外的功能，它將在這裡添加。

      https://github.com/nwbb/node-brightcove

    6. 播放程式管理 API 用戶端

      此模組可用於存取布萊特灣播放器管理 API。基礎 API 的文檔可以在這裡找到。

      https://github.com/BrightcoveLearning/Player-Management-Client

    7. 使用 CSRP 的布萊特灣 API 示例

      這個項目顯示了如何新的布萊特灣 API 可以與 C＃ 一起使用。

      請注意，這將從桌面環境（測試過）工作，但由於 CORS，如果在 ASP.NET 頁面中使用，可能無法正常工作。如果您遇到此問題，我相信解決此問題的唯一方法是在所有用戶中使用Brightcove提供的PHP代理。CMS API示例頁面。

      所有代碼都按原樣提供，純粹作為例子。使用它由您自己承擔風險。不提供任何形式、形狀或形狀的支撐。

      https://github.com/mikelau13/BrightcoveAPI

    8. 播放程式管理 API 用戶端

      此模組可用於存取布萊特灣播放器管理 API。基礎 API 的文檔可以在這裡找到。

      https://github.com/BrightcoveLearning/Player-Management-Client

    播放器和 SDK

    1. Video.js 外掛程式架構產生器

      這是一個自以為的約曼生成器的 video.js 插件。它基於 video.js 核心團隊的建議以及在 Brightcove 開發插件的工具和慣例。

      https://github.com/videojs/generator-videojs-plugin

    系統整合

    1. 布萊特灣視頻連接

      SharePoint 2013 的布萊特灣視頻連接允許發布商直接從 SharePoint 添加，瀏覽和管理視頻和視頻播放列表。它的播放器組件允許輕鬆地將 Brightcove 視頻雲資產直接集成到 SharePoint 頁面。

      http://go.brightcove.com/bc-video-connect-sharepoint2013

    2. 適用於 Drupal 的布萊特灣視頻連接

      Brightcove 視頻雲是一個視頻託管和發布解決方案，用於在任何設備上向數字觀眾提供專業質量的視頻。它提供了高度可定制的播放器，高清流，與第三方 CDN 和 YouTube 同步。與 Drupal 的 Brightcove 視頻連接，您可以有效地添加和管理您的 Drupal 支持的網站上的視頻內容。

      https://www.drupal.org/project/brightcove

    3. 白海灣視頻連接

      您是否正在尋找在 WordPress 中原生處理您的 Brightcove 影片和播放清單資料庫？有了這個插件, 由 10up.com 開發, 你有權處理多個帳戶和視頻庫, 上傳視頻並將它們添加到播放列表, 渲染短碼與您的視頻和所有從 WordPress 管理界面.

      https://wordpress.org/plugins/brightcove-video-connect/

    4. 連接器

      提供多個視訊雲端帳戶之間的整合。

      https://github.com/brightcove/sharepoint-connector

    5. 適用於 Adobe 體驗管理員的布萊特灣視訊連線

      適用於 Adobe 體驗管理員 (AEM) 的布萊特灣視訊連線是整合布萊特灣視訊雲端和 AEM 之間工作流程的最簡單方式。使用 Brightcove 視訊連線，AEM 使用者可以直接從 AEM 使用者介面中新增、瀏覽和管理視訊。

      http://go.brightcove.com/bc-video-connect-aem

    6. CQ5 元件

      CQ5 元件整合了布萊特灣

      技術：CQ5

      https://www.github.com/coresecure/Adobe-CQ5-Brightcove-Connector

    7. 推銷鏟車

      Salesforce 提供大量 API，可簡化從資料存放區到銷售人員的大量資料同步。該項目提供了一個寶石在紅寶石應用程序中使用 Salesforce 批量 API。

      技術：銷售員，紅寶石

      https://www.github.com/brightcove/salesforklift

    分析

    1. Adobe Analytics

      您可以使用播放器模組輕鬆設定 Adobe Analytics 外掛程式。有關信息，請參見配置Adobe Analytics事件跟踪。

    2. Google Analytics

      谷歌分析插件可以很容易地使用球員模塊進行配置。有關信息，請參見配置Google Analytics（分析）事件跟踪。

    雜

    這些項目不適合任何其他類別。

    1. Perl 共享資源

      該項目提供了通用的類和實用程序在其他 Perl 項目中使用。這可能包括第三方圖書館以及 Brightcove 開放原始碼庫。

      技術：Perl

      https://www.github.com/BrightcoveOS/Perl-Commons

    2. 度量石墨泡菜

      科達海爾的擴展

      技術：石墨, 爪哇

      https://www.github.com/BrightcoveOS/metrics-graphite-pickle

    3. LG 範例應用程式

      這是一個 LG 視頻應用程序，顯示來自 Brightcove 帳戶的幾個播放列表。該應用程序的設計非常簡單，並允許客戶根據他們想要的功能進行自定義來更改 UI 或添加/刪除功能。

      技術：LG, 電視

      https://www.github.com/BrightcoveOS/LG-Sample-App

    4. Roku 範例應用程式

      這個範本提供了一個快速的開始，讓您的 Brightcove 內容發佈在 Roku 上。若要開始使用此應用程式發行，您需要下列項目：

      技術：六，電視

      https://www.github.com/BrightcoveOS/Roku-Sample-App

    5. Java 共享資源

      該項目提供了在其他 Java 項目中使用的通用類和實用程序。這可能包括第三方圖書館以及 Brightcove 開放原始碼庫。

      技術：爪哇

      https://www.github.com/BrightcoveOS/Java-Commons

    6. 鑽石

      鑽石是一個蟒蛇守護進程，收集系統指標並將它們發布到石墨。它能夠收集 CPU，內存，網絡，I/O，負載和磁盤度量。此外，它還提供 API，用於實作自訂收集器，以便從幾乎任何來源收集度量。

      技術：石墨、蟒蛇

      https://github.com/python-diamond/鑽石
