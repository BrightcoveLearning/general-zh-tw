Oooyala 視訊雲端擷取適配器：清單文件格式

本主題涵蓋您可以用來擷取視訊和資產的資訊清單檔案格式。
注意：本主題適用只要 Ooyala 客戶遷移到視頻雲。

簡介

擷取影片時，您可以使用資訊清單檔案來指定額外的中繼資料 (包括影片標題、說明、飛行時間、標籤和自訂中繼資料)，以及縮圖和隱藏式字幕檔案。資訊清單檔案的格式為 XML 或 CSV 格式。如需資訊清單的詳細資訊，請參閱：

  • CSV清單文件
  • XML清單文件

處理開始時

筆記：本主題適用於 OIS v2.5。
筆記：在收到 XML 或 CSV 資訊清單檔案之前，視訊雲端不會開始處理任何視訊。因此，選擇批次上傳策略非常重要。如果您的影片必須快速取得 (例如新聞發佈)，您應該建立非常小批次，甚至是個別批次 (每部影片一個 CSV 或 XML 檔案)。
