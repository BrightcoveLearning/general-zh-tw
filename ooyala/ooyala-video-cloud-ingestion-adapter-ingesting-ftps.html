Ooyala-視頻雲提取適配器：使用 FTPS 攝取

您可以繼續使用 FTP 搭配 XML 資訊清單，將視訊內嵌到視訊雲端。本主題提供程序的概觀。
注意：本主題適用只要 Ooyala 客戶遷移到視頻雲。

簡介

您可以使用 FTPS (建議使用) 或 FTP (已停用)，將影片、縮圖、隱藏式字幕和資訊清單檔案上傳至 Video Cloud。

筆記：本主題適用於 OIS v2.5。

FTP 內嵌概觀

關於 FTPS

FTP 是標準的網路通訊協定，用於將檔案從一台電腦傳輸到另一台電腦。對於 OIS 版 2.5，必須使用 FTP (透過 SSL 進行 FTP)。大多數作業系統本身支援 FTP，您可以從許多 FTP 用戶端應用程式中選擇。您可以通過編寫腳本自動化 FTP 上傳，讓用戶通過 Windows，Mac 或 Linux 應用程序批量上傳視頻，或者讓用戶通過瀏覽器（例如，FirefTP）批量上傳視頻。

筆記：您可以使用任何 FTP 用戶端上傳影片。

帳號策略

您的帳戶隨時限制在擷取伺服器上的最大 100GB。如果您希望上傳更多內容，您可以向客戶成功經理或 Brightcove 支援要求暫時增加。

使用 FTPS 用戶端

您可以透過 FTP /FTPS 上傳影片，無論是否有資訊清單。

筆記：所有上傳文件都必須符合支持的提取格式

透過 FTPS (OIS v2.5 版) 上傳您的檔案

筆記：OIS 2.5 版需要 FTPS 用戶端。
  1. 準備您要上傳的檔案。
  2. 啟動您的 FTPS 用戶端程式。
  3. 使用僅上傳 Video Cloud 使用者帳戶的認證 (電子郵件和密碼) 登入內嵌端點。
  4. 上傳您想要內嵌的檔案 (影片、縮圖、隱藏式字幕檔案等)。如果您使用的是資訊清單檔案，請務必上傳資訊清單檔案中參考的所有檔案。
  5. Oooyala 建議你最後上傳清單文件。

檔案在處理所有檔案後立即刪除。未完成整個擷取工作流程的已提交檔案 (例如，等待上傳資訊清單檔案) 會在 7 天後從伺服器中刪除。

筆記：透過 FTP 內嵌檔案並使用資訊清單式擷取時，不支援檔案名稱中的特殊字元 (非 ASCII)。
